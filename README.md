# Stream Beta
## Plataforma de videos e streamers

## Requisitos:

    NodeJs e Yarn

## Ruby version

    ruby '3.0.2'

## Setup:

    bin/setup

## Criação do banco de dados

    bin/rails db:migrate

## Seeds
